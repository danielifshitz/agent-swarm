## Two-agent collaboration

When `.swarm/` exists, the user asks you to work with another AI agent/session,
or unread mail exists in `.swarm/mailbox`, read `.swarm/PROTOCOL.md` completely
before task work. Then read `.swarm/ROSTER.md`, `.swarm/MODE`, and
`.swarm/TASK.md`. Never guess your agent id; ask if it was not explicitly
assigned. Use `.swarm/swarm` for all mailbox operations.
