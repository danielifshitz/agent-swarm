## Two-agent collaboration

If the user asks to initialize, prepare, or configure a run or asks for the two
launch prompts, follow `.agents/skills/swarm-init/SKILL.md` and do not assume an
agent identity.

Otherwise, when the user assigns agent-a or agent-b or unread mail exists in
`.swarm/mailbox`, read `.swarm/PROTOCOL.md` completely before task work. Then
read `.swarm/ROSTER.md`, `.swarm/MODE`, and `.swarm/TASK.md`. Never guess your
agent id; ask if it was not assigned. Use `.swarm/swarm` for all mailbox work.
